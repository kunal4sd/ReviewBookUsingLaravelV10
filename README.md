## Book Review Project laravel 10

Application to add Review to the book and show rating

<ul>Used Technologies:
    <li>Laravel 10</li>
    <li>PHP 8.2</li>
    <li>MySQL</li>
    <li>Alpine.js</li>
    <li>TailwindCSS</li>
</ul>
<ul>Laravel Features:
    <li>Routes</li>
    <li>Models</li>
    <li>Blade Templates</li>
    <li>Pagination</li>
    <li>Factories and Seeder to generate data</li>
    <li>CRUD</li>
    <li>Flash Messages</li>
    <li>Rate Limiting</li>
    <li>Migrations</li>
    <li>OneToMany Relationship</li>
    <li>Controllers</li>
    <li>Caching</li>
    <li>Resource Controller</li>
    <li>Component</li>
</ul>
To setup project run in CLI:

Clone Project:

```sh

git clone https://github.com/kunal4sd/ReviewBookUsingLaravelV10.git

```

Go inside project directory:

```sh

cd ReviewBookUsingLaravelV10

```

```sh

composer install

```

```sh

cp .env.example .env

```

```sh

php artisan key:generate

```

```sh

php artisan migrate

```

after migration command select yes option to create database schema

run application:

```sh

php artisan serve

```
